File I/O with OSIC

Nov 1, 2018

OSIC provides simple tools to write data to files and read files. Here you can easily use commands and grant complex rights. The technical basis is provided by the POSIX libs and can be derived functionally from them.

To write a file it is enough to open, write and close the file:

Writing and reading files is part of libosic.so. With the flag 2 in os.open() we guarantee the read/write access.

os.write() returns a boolean value about the success of its operation. In this snippet a file is read (plaintext):

The creation of a file, can simple done by Flag 2.

But you are able to specifiy the file rights. This can be done by the full POSIX command. os.open('foobar.txt', 100 , 655) - the 100 represents the integer representation of O_CREATE and 655 the chmod of the new file. Please not, that this is not working on Windows - there you use the flags- given on the example above.

Also can the rights lead you to a dead end.
